# Taschenrechner

Ein einfacher Taschenrechner geschrieben in Python.

## Funktionen

- Addition
- Subtraktion
- Multiplikation
- Division
- Fehlerbehandlung bei falschen Eingaben
- Verlauf speichern
- Gespeicherten Verlauf anzeigen

## Aufbau
main.py - Hauptprogramm

rechner.py - Rechenlogik

eingaben.py - Benutzereingaben

speichern.py - Speichern des Verlaufs

## Starten

Python 3 benötigt.

Programm starten mit:

python main.py
